## About

BrewMate is a native macOS menu bar app that helps you manage your Homebrew services with ease. Start, stop, and monitor services like PostgreSQL, Redis, MySQL, nginx, and more — all from your menu bar.

## Features

- **Menu Bar Access** — Quick access to all your Homebrew services
- **One-Click Control** — Start, stop, and restart services instantly
- **Service Status** — Real-time status monitoring
- **Favorites** — Pin frequently used services for quick access
- **Groups** — Organize services into custom groups
- **Dependencies** — View and manage service dependencies
- **Native macOS** — Built with SwiftUI, feels right at home

## Installation

1. Download `BrewMate.dmg` from the link above
2. Open the DMG file
3. Drag BrewMate to your Applications folder
4. Launch BrewMate from Applications
